
# File Manager

## Installation

```bash
$ git clone git@github.com:RB-Lab/file-manager.git
$ npm i
```

## Running the project

### Running the mock API

```bash
$ npm run mock-api
```

### Running project in development mode

```bash
$ npm start
```

### Building project
If you prefer to just build without the live reload and build-on-each-change watcher, run:

```bash
$ npm run build
```
